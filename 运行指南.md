# AI潜能测评系统 - 运行指南

## 🚀 快速开始

### 方式1：本地运行（推荐）

1. **下载代码**
   ```bash
   git clone https://github.com/harrysonghuahua/ai-assessment-system.git
   cd ai-assessment-system
   ```

2. **安装依赖**
   ```bash
   pip3 install -r requirements.txt
   ```

3. **配置API密钥**
   - 复制 `.env.example` 为 `.env`
   - 在 `.env` 中配置智谱AI API密钥：
   ```
   ZHIPU_API_KEY=你的API密钥
   ```

4. **运行应用**
   ```bash
   python3 app.py
   ```

5. **访问应用**
   - 打开浏览器访问：`http://localhost:8000`

### 方式2：Docker运行

1. **构建镜像**
   ```bash
   docker build -t ai-assessment .
   ```

2. **运行容器**
   ```bash
   docker run -p 8000:8000 ai-assessment
   ```

3. **访问应用**
   - 打开浏览器访问：`http://localhost:8000`

### 方式3：云服务器部署

1. **购买云服务器**（阿里云/腾讯云学生优惠）
2. **上传代码到服务器**
3. **安装环境**
   ```bash
   sudo apt update
   sudo apt install python3 python3-pip
   pip3 install -r requirements.txt
   ```

4. **运行应用**
   ```bash
   python3 app.py --host=0.0.0.0 --port=80
   ```

## 📋 系统要求

- Python 3.7+
- 现代浏览器（Chrome/Firefox/Safari）
- 网络连接（用于AI API调用）

## 🔑 API密钥获取

### 智谱AI API
1. 访问：https://open.bigmodel.cn/
2. 注册账号并登录
3. 在控制台获取API密钥
4. 在 `.env` 文件中配置

## 🛠️ 故障排除

### 常见问题

**问题1**：端口被占用
```bash
# 查找占用端口的进程
lsof -ti:8000 | xargs kill -9
```

**问题2**：依赖安装失败
```bash
# 使用国内镜像源
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**问题3**：API调用失败
- 检查 `.env` 文件中的API密钥
- 检查网络连接
- 查看控制台错误信息
